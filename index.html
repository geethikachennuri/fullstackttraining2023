Markup language
The “Markup Language” component of HTML refers to the insertion of instructions, called tags. Tags tell the Web browser how to interpret the data. Tags follow a standard format. Each tag begins with a “less than” symbol (<), immediately followed by the tag text, and ending in a “greater than” symbol (>)
